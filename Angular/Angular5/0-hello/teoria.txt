Crear una aplicacion con preparada para routing:
ng new "nombre" -p "prefijo" --routing true

Servir la app:
npm start

Para crear un componente:
ng g c "nombre_componente"

Si estamos en el modulo de entrada de la app y queremos que salga en el index.html de inicio, 
debemos incluir el componente en el decorador del modulo, en el array de bootstrap.

Ejemplo: 
crear component: 
$ ng g c HeaderComponent

llamar y declarar componente:

index.html 
<ca-header></ca-header>

app.module.ts
import { HeaderComponent } from './header/header.component';

@NgModule({
  declarations: [
    AppComponent,
    HeaderComponent
  ],
  imports: [
    BrowserModule,
    AppRoutingModule
  ],
  providers: [],
  bootstrap: [AppComponent, HeaderComponent]
})   
// En todos los componentes pertenecientes al modulo AppModule estara disponible el HeaderComponent
